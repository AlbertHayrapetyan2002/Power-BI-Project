This is my Power BI educational project

# 📊 Sales Performance Dashboard | Power BI

Интерактивный аналитический дашборд для мониторинга ключевых бизнес-показателей: выручки, прибыли, транзакций и возвратов.

Проект демонстрирует навыки работы с моделью данных, DAX-мерами, KPI-метриками и визуальной аналитикой в Power BI.

---

## 🎯 Цель проекта

Создать аналитический отчёт для:
- Отслеживания динамики продаж во времени
- Сравнения показателей с предыдущим месяцем
- Анализа прибыльности
- Контроля возвратов
- Географического анализа продаж
- Оценки эффективности брендов

---

## 🧠 Бизнес-вопросы, которые решает дашборд

- Как изменилась выручка по сравнению с прошлым месяцем?
- Растёт ли прибыль?
- Какой уровень возвратов?
- Какие бренды приносят наибольшую прибыль?
- В каких регионах больше всего транзакций?
- Достигаем ли мы целевого уровня выручки?

---

## 🗂️ Структура модели данных

### Fact Table:
- `Transactions`
- `Returns`

### Dim Tables:
- `Calendar`
- `Products`
- `Stores`
- `Customers`
- `Regions`

### Таблица мер:
- `All Measures`

Модель построена по принципу **Star Schema (звёздная схема)**.

---

## 📈 Основные метрики (DAX)

| Метрика | Описание |
|----------|----------|
| Total Revenue | Общая выручка |
| Revenue Target | Целевая выручка |
| Total Transactions | Количество транзакций |
| Total Profit | Общая прибыль |
| Profit Margin | Маржинальность |
| Total Returns | Количество возвратов |
| Return Rate | Процент возвратов |
| Last Month Revenue / Profit / Transactions | Показатели прошлого месяца |

---

## 📊 Визуализации

- KPI карточки (сравнение с прошлым месяцем)
- Gauge (выполнение плана по выручке)
- Column Chart (динамика по неделям)
- Map (география транзакций)
- Treemap (структура продаж по регионам)
- Matrix (анализ по брендам)
- Slicer (фильтр по странам)
